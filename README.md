# Word Gacha · 단어 가챠

초/중등 학생 영단어 학습 앱 모바일 목업.
캐릭터를 키우고 도감을 모으는 게임화 디자인.

**Live**: https://sigongjoa.github.io/word-gacha-mockup/

## 디자인 방향

- 캐릭터(토토)가 화면을 지배하는 양산형 카와이 패턴 (Pokemon Sleep / Finch / Duolingo 참조)
- 4-layer idle 애니메이션: 호흡 · 눈깜빡 · 시선 추적 · 귀 움직임
- 표정 3종 (happy / hungry / sleepy)
- 4개 화면 IA (홈 · 미션 · 도감 · 나)
- 학생 친화 폰트: Jua + Bagel Fat One + Fredoka

## 기술

- 정적 단일 HTML, no build
- prefers-color-scheme 다크모드
- prefers-reduced-motion 처리
- safe-area-inset (iOS 노치)
- WCAG AA 대비
